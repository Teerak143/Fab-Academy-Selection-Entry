Adaptive Learning Modules

The Adaptive Learning Modules project is a physical "smart tutor" designed to help students learn through hands-on interaction. Instead of a boring paper quiz or a screen, this is a tactile machine where students solve puzzles or answer questions using levers, sliders, and buttons.

The core of the project is an adaptive difficulty engine. If a student struggles with a specific logic puzzle or math problem, the machine recognizes the pattern of errors and physically changes the next task to be slightly easier—perhaps by lighting up a hint via LEDs or simplifying a mechanical lock. Once the student succeeds, the machine "levels up" the difficulty again.
